# Start Here

This workshop is designed to be completed in about ~50 minutes. You do not need to write code, but you do need to move quickly and stay on the numbered path.

## Before you begin

0. An instructor will pair you up.
Follow the instructor through:
1. Create an IBM Cloud account:
    - Go to https://cloud.ibm.com/registration?utm_content=academicsb
    - Register for a Free IBM Cloud Account
    - Make sure you use an email account you have access to
2. Provision watsonx orchestrate instance (Free Tier)


Have these ready before following the instructions in this repo:
- An IBM watsonx Orchestrate account you can log into
- This [website](https://github.com/ianre/juntos-ibm-workshop)  open on your laptop
